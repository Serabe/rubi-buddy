# RubiBuddy

A small Ruby class for accessing the [i-Buddy](http://www.i-buddy.com/)'s functionality with Ruby.

See ''examples/'' folder.

## Requirements

  - Ruby >= 1.8.6
  - libusb 
  - ruby-usb: http://www.a-k-r.org/ruby-usb/ (have don't have tried with the gems)
